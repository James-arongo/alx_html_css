What is HTML

HTML stands for HyperText Markup Language. It is the language used to write documents which users see on web browsers.

HyperText

Hypertext is what allows developers of web pages to create links that enable navigation from one web page to another.

Markup

A markup language allows for the use of text annotations to provide additional meaning to the contents of a document. HTML achieves this using tags(also known as markup). Tags provide meaning as to how content on the document should be displayed. Browsers use tags to show the appropiate structure for the web page.

We use two categories of tags to structure pages:

  • Structural markup: elements used to describe the structure of the document. Example elements: headings (<h1> through <h6>), and <p> elements.
  • Semantic markup: elements which add extra information to the web page but do not affect the structure of the page. Their purpose is so that other programs like screen readers and search engines can use them and know what their function will be. For example, the voice of a screen reader may add emphasis to text around an <em> element.
